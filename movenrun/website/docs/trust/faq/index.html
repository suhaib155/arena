Frequently asked questions

Direct questions, direct answers — with links to the pages that carry the detail.

Is MovenRun live?

The marketing website is live, and a development-stage Android beta exists (built through an APK pipeline). There is no public app-store release. The full territory experience currently runs as an on-device preview — see Current status.

Is a wallet required?

No. The app contains no wallet connection at all. The core experience will never require one — that is an architectural rule, not a temporary state.

Is MovenRun move-to-earn?

No. There is no earning mechanic and no token payout. MovenRun deliberately avoids tying motivation to token value — that model has repeatedly harmed the users it attracted. Movement earns in-app progress: territory, streaks, XP.

Can I earn money with MovenRun?

No. Nothing in the product is redeemable, tradable, or valuable, and no future earnings are promised. If that ever changes, it will happen through audits, legal review, and public release gates — announced plainly, not hinted.

Do Zone Deeds represent real land?

No. A Zone Deed is a digital game record tied to a map cell. It has no connection to real-world property or legal title — see Zone Deeds.

What is currently available?

GPS movement sessions, route summaries, XP/streak progression, weekly recaps — implemented on-device — plus the full territory loop (capture, defense, clubs, districts, city layer, deed showroom) as labeled local previews. The single source of truth is the status matrix.

What is “local preview” data?

Data generated on your device — deterministic simulations and fictional entities (clubs, rivals, sponsors, events) — used to make future systems tangible today. It involves no servers, no other players, and no real organizations.

What is Base Sepolia?

A public test network for the Base blockchain, where developers deploy experiments with valueless test tokens. MovenRun's V1 contracts live there — and only there. See Contract status.

Is V2 deployed?

No. The V2 contract suite exists as source code and tests in an open pull request — under review, not merged, not deployed, not audited.

How is my GPS data handled?

Today: foreground-only tracking, all state on your device, and derived features keep summaries rather than raw trails. Nothing is uploaded. Details and future plans: Privacy & movement data.

When will mainnet launch?

No date exists, and none is promised. Mainnet anything is conditional on the release gates: verification reliability, density, retention, independent audit, legal review, and more. It may not happen at all — the game is designed to be complete without it.

How can I follow the project?

Watch the GitHub repository, follow on X, and check the development history page, which is updated as milestones land.
