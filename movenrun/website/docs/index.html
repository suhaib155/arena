MovenRun documentation

MovenRun is an onchain movement network. You move through the real world — walking, running, cycling — and the ground you cover becomes territory you verifiably own, trade, and govern. It runs on Base, and it is built so anyone can play in seconds, with no seed phrases, no gas, and no crypto experience required.

In one paragraph. Every session turns real movement into a cryptographic proof of activity that settles onchain. That proof lets you capture map zones, hold them as Zone Deeds you truly own, and earn $MOVE — the asset that powers the whole economy. Clubs contest districts, cities run seasons, and the network itself is designed to be owned and governed by the people who move on it.

Why MovenRun is different

You own what you coverTerritory isn't a score in a database — it's an onchain asset with your name on it. Movement you can proveReal activity becomes a verifiable proof, so the map rewards effort, not spoofing. Crypto that disappearsA wallet is created for you invisibly. No gas, no jargon — it just feels like a game.

The core loop

  1. Move — walk, run, or cycle. Your route becomes a verifiable proof of movement.
  2. Capture — the zones you cross become claimable territory on a global hex map.
  3. Own — claimed zones become Zone Deeds: onchain assets you hold, defend, and can transfer.
  4. Grow — territory compounds into districts, clubs, seasons, $MOVE, and a vote in how the network runs.

Built for everyone

MovenRun is fully onchain, but you would never know it unless you looked. Signing up takes a tap; a secure wallet is created for you in the background; transactions are sponsored so there is no gas to buy; and the interface speaks the language of a movement game, not a blockchain. Crypto-native players get real ownership and composability. Everyone else just gets a great game — that happens to make them true owners. See Onchain, made invisible.
